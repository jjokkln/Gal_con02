# OpenAI API Configuration
# ========================
# 
# Erhalte deinen API Key von: https://platform.openai.com/api-keys
# 
# Für lokale Entwicklung:
# 1. Kopiere diese Datei und benenne sie um zu: .env
# 2. Füge deinen echten API Key unten ein (ersetze "your-api-key-here")
# 3. Die .env Datei wird automatisch von der App geladen
#
# Für Streamlit Cloud Deployment:
# 1. Gehe zu: App Settings → Secrets
# 2. Füge folgende Zeile hinzu:
#    OPENAI_API_KEY = "your-api-key-here"
# 3. Ersetze "your-api-key-here" mit deinem echten Key
# 4. Speichere die Secrets
#
# WICHTIG: 
# - Teile deinen API Key NIEMALS öffentlich!
# - Die .env Datei ist in .gitignore und wird nicht gepusht
# - Für Streamlit Cloud MÜSSEN die Secrets manuell eingegeben werden

OPENAI_API_KEY=your-api-key-here
